# Quick Start Guide - Celeste AI

## 🚀 Get Started in 5 Minutes!

### 1. Install Python
Download and install Python 3.8+ from [python.org](https://www.python.org/downloads/)

### 2. Download Celeste
```bash
git clone https://github.com/yourusername/celeste-ai.git
cd celeste-ai
```

### 3. Install Dependencies
```bash
pip install -r requirements.txt
```

### 4. Get Gemini API Key
1. Go to [Google AI Studio](https://makersuite.google.com/app/apikey)
2. Create API Key
3. Copy the key

### 5. Configure
Create `.env` file:
```env
GEMINI_API_KEY=paste_your_key_here
```

### 6. Run Celeste!
```bash
python main.py
```

## 🎤 First Commands to Try

After starting Celeste, choose text mode (`T`) and try:

1. `what time is it`
2. `tell me the weather`
3. `lets talk about space`
4. `search for python programming`
5. `open gmail`

## 📝 Optional Setup

### For Email Sending:
Add to `.env`:
```env
EMAIL_ADDRESS=your_email@gmail.com
EMAIL_APP_PASSWORD=your_app_password
```
Get app password: [myaccount.google.com/apppasswords](https://myaccount.google.com/apppasswords)

### For WhatsApp:
Install WhatsApp Desktop from [whatsapp.com/download](https://www.whatsapp.com/download)
